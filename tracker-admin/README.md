# Kerala Election Tracker - Admin Dashboard

A comprehensive admin dashboard for managing and analyzing Kerala election data. This Next.js application provides tools for managing election booths, candidates, constituencies, and performing detailed electoral analysis.

## Project Overview

The Kerala Election Tracker Admin Dashboard enables administrators to:

- **Manage Election Data**: Handle candidate mappings, constituency configurations, and booth assignments
- **Booth Management**: Oversee booth managers and vote data collection across districts
- **Local Body Administration**: Configure ward-assembly mappings and manage local body elections
- **Data Analysis**: Generate alliance analysis, swing analysis, and constituency-level insights
- **CSV Data Import**: Bulk upload and manage election data through CSV files
- **Electoral Insights**: Analyze voting patterns and performance metrics across regions

## Getting Started

### Prerequisites

- Node.js 16.x or higher
- npm or yarn package manager

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to access the admin dashboard.

### Build

Create an optimized production build:

```bash
npm run build
```

### Production

Start the production server:

```bash
npm start
```

### Linting

Check code quality:

```bash
npm run lint
```

## Project Structure

- `app/` - Next.js app directory with pages and API routes
- `components/` - React components for admin dashboard tabs and features
- `lib/` - Utility functions and API helpers
- `config/` - Configuration files and environment settings
- `public/` - Static assets

- [Next.js](https://nextjs.org) - React framework
